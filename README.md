# 土佐日記をたどる

紀貫之『土佐日記』に出てくる場所・情景・文章に表れた気持ちを、いまの高知でたどる地域文化PRサイトです。
「1000年前の旅日記を、今の高知でたどってみよう。」をテーマに、帰京の海路と、和歌にこめられた心を紹介します。

## 公開URL

https://ユーザー名.github.io/tosa-nikki/

※GitHub Pages を有効にすると発行されるURLに書き換えてください。

## 内容・構成

- **『土佐日記』とは** … 約1100年前に紀貫之が書いた、日本最古の仮名日記のやさしい紹介
- **海の道をたどる** … 帰京の旅（大津・浦戸 → 大湊 → 奈半 → 羽根 → 室津 → 室戸岬 → 京）を、現在の高知の地名とGoogleマップのリンクでたどるタイムライン
- **本音が、見える** … 望郷の歌や、土佐で亡くした娘を思う和歌など、文章に表れた心を縦書きで紹介
- **問いかけ** … 読んだ人へのメッセージ

## 使用技術

- HTML / CSS / JavaScript（フレームワークなし、1ファイル構成）
- 縦書き（writing-mode）、スクロール連動アニメーション（IntersectionObserver）
- フォント：Shippori Mincho B1 / Noto Serif JP（Google Fonts）

## 制作について

高校の探究学習「地域の文学を知る『土佐日記』探訪」の課題として制作した学習用の作品です（非公式）。
地名・旅程・和歌の解釈は諸説ある場合があります。
